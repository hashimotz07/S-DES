Este projeto implementa o algoritmo {\bfseries{S-\/\+DES (Simplified DES)}}, uma versão didática do algoritmo de cifra de blocos DES, como parte do Trabalho de Implementação 1 da disciplina {\bfseries{Segurança Computacional – CIC0201 (UnB)}}.

\DoxyHorRuler{0}
\hypertarget{md_README_autotoc_md2}{}\doxysection{🎯 Objetivo}\label{md_README_autotoc_md2}
A implementação tem como foco\+:


\begin{DoxyItemize}
\item Realizar {\bfseries{cifragem e decifragem}} de blocos de 8 bits com uma chave de 10 bits.
\item Utilizar o algoritmo S-\/\+DES completo, incluindo\+:
\begin{DoxyItemize}
\item Geração das subchaves K1 e K2
\item Permutações e trocas (P10, P8, IP, IP⁻¹, etc.)
\item Rodadas de Feistel com S-\/\+Boxes
\end{DoxyItemize}
\item Suportar os {\bfseries{modos de operação ECB e CBC}}, com vetores de inicialização.
\end{DoxyItemize}
