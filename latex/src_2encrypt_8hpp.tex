\doxysubsubsection{\texorpdfstring{encriptacao()}{encriptacao()}}
{\footnotesize\ttfamily int encriptacao (\begin{DoxyParamCaption}\item[{int}]{n,  }\item[{int}]{k }\end{DoxyParamCaption})}



Realiza a encriptação de um bloco de 8 bits usando S-\/\+DES. 

O comportamento depende do valor global \textquotesingle{}encriptar\textquotesingle{}\+:
\begin{DoxyItemize}
\item Se encriptar == 2\+: Encripta com K1 → K2 (padrão)
\item Se encriptar == 1\+: Decripta com K2 → K1
\end{DoxyItemize}


\begin{DoxyParams}{Parâmetros}
{\em n} & Bloco de 8 bits a ser encriptado (0-\/255) \\
\hline
{\em k} & Chave mestra de 10 bits (0-\/1023) \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Retorna}
int Bloco encriptado de 8 bits
\end{DoxyReturn}
\begin{DoxyPrecond}{Pré-\/condição}
n deve ser um valor de 8 bits (0 $<$= n $<$= 255) 

k deve ser um valor de 10 bits (0 $<$= k $<$= 1023) 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Pós-\/condição}
O resultado será um bloco de 8 bits encriptado
\end{DoxyPostcond}
\begin{DoxyNote}{Observação}
O processo segue estas etapas\+:
\begin{DoxyEnumerate}
\item Geração das subchaves K1 e K2 -\/$>$ Inverte a ordem se encriptar == 1
\item Permutação inicial (IP)
\item Aplicação da rede de Feistel (2 rodadas)
\item Permutação final (IP⁻¹) 
\end{DoxyEnumerate}
\end{DoxyNote}
