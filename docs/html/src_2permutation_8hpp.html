◆ permuta()

int permuta ( int  n,
vector< int > &  v,
int  total_bits = 10 
)

Aplica uma permutação em um número inteiro usando uma tabela específica.

Cada bit do resultado é mapeado a partir da tabela de permutação.

Parâmetros
nNúmero de entrada (bits a serem permutados).
vTabela de permutação (ex.: p10, p8, ip, etc.).
total_bitsNúmero total de bits em 'n' (padrão: 10).
Retorna
int Valor permutado com tamanho igual ao tamanho da tabela 'v'.
Pré-condição
v não deve estar vazia.
total_bits deve ser >= ao maior valor em 'v'.
n deve ser um valor válido dentro do número de bits especificado.
Pós-condição
O resultado terá exatamente v.size() bits.
