Converte um inteiro para sua representação binária em string

Parâmetros
nNúmero inteiro a ser convertido
szTamanho (em bits) da saída desejada
Retorna
string Representação binária com exatamente 'sz' caracteres ('0' ou '1')
Pré-condição
sz deve ser maior que 0 e suficiente para representar 'n'

toBin(5, 4) retorna "0101"
