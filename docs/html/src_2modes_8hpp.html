Decripta texto cifrado usando modo CBC.

Parâmetros
cipherTextVetor de strings binárias (blocos cifrados de 8 bits)
chaveChave mestra de 10 bits usada na encriptação
IVMesmo vetor de inicialização usado na encriptação
Retorna
string Texto plano reconstruído
Aviso
O IV deve ser o mesmo usado na encriptação
Observação
Processo:
  1. Adiciona IV no início do fluxo
  2. Processa blocos em ordem reversa
  3. Aplica XOR com bloco anterior cifrado

Encripta texto plano usando modo CBC (Cipher Block Chaining)

Parâmetros
plainTextVetor de strings binárias (blocos de 8 bits)
chaveChave mestra de 10 bits (0-1023)
IVVetor de inicialização (8 bits) para o primeiro bloco
Retorna
string Texto cifrado concatenado
Pré-condição
IV deve ser valor de 8 bits (0-255)
Observação
Características do CBC:
  • Blocos idênticos produzem saídas diferentes
  • Propagação de erros entre blocos
  • Requer IV único por encriptação

◆ ECB_encripta()

string ECB_encripta ( vector< string > &  plainText,
int  chave 
)

Encripta plaintext usando o modo ECB (Electronic Codebook)

Parâmetros
plainTextVetor de strings binárias (blocos de 8 bits)
chaveChave mestra de 10 bits (0-1023)
Retorna
string Texto cifrado concatenado
Pré-condição
Cada string em plainText deve ter tamanho 8 e conter apenas '0's e '1's
chave deve ser um valor de 10 bits válido
Pós-condição
O resultado terá tamanho plainText.size() * 8
Observação
Características do ECB:
  • Blocos idênticos produzem saídas idênticas
  • Não usa vetor de inicialização (IV)
  • Paralelizável
