Controle global de operação:
  • encriptar = 2: Modo encriptação (usa K1 depois K2)

◆ encriptacao()

int encriptacao ( int  n,
int  k 
)

Realiza a encriptação de um bloco de 8 bits usando S-DES.

O comportamento depende do valor global 'encriptar':

  • Se encriptar == 2: Encripta com K1 → K2 (padrão)
  • Se encriptar == 1: Decripta com K2 → K1
Parâmetros
nBloco de 8 bits a ser encriptado (0-255)
kChave mestra de 10 bits (0-1023)
Retorna
int Bloco encriptado de 8 bits
Pré-condição
n deve ser um valor de 8 bits (0 <= n <= 255)
k deve ser um valor de 10 bits (0 <= k <= 1023)
Pós-condição
O resultado será um bloco de 8 bits encriptado
Observação
O processo segue estas etapas:
  1. Geração das subchaves K1 e K2 -> Inverte a ordem se encriptar == 1
  2. Permutação inicial (IP)
  3. Aplicação da rede de Feistel (2 rodadas)
  4. Permutação final (IP⁻¹)
